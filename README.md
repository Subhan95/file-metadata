#File Metadata

An API that returns a given file's size in bytes

##Getting Started

###Prerequisites

1. [node](https://nodejs.org/en/)
2. [npm](https://www.npmjs.com)

###Installing
Clone the project

```git
git clone https://github.com/khansubhan95/request-header-parser.git
```

run

```
npm install
```

to install the dependencies
